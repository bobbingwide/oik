To display a Google Map for your company location

[bw_show_googlemap]

Some of the default values are extracted from oik information:

  • company - for the Company name
  • lat - for the latitude
  • long - for the longitude
  • width - map width ( 100% )
  • height - map height ( 400px - to allow for the info window )
